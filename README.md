### Systems Philosophy
* **<font color="#FF8C00">Grueling by Design:</font>** Every mechanic is built to be a challenge.
* **<font color="#FF8C00">Modular Design:</font>** Development guidelines adopted from the ACE standard.
* **<font color="#FF8C00">FOSS Commitment:</font>** The project's code is a resource. Fork it, learn from it, break it.
